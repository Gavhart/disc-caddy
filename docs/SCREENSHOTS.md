# Store & marketing screenshots

Capture these from the **running app** (not mockups) for App Store, Play Store, and `/invite`.

## Setup

1. Run `npm run dev` and sign in with a Pro account that has a bag, rounds, and a league.
2. Use a desktop browser at **390×844** (iPhone) or **1080×1920** for phone frames.
3. Save PNGs to `store-assets/screenshots/` with the names below.

## Required flows

| File | Route | What to show |
|------|-------|----------------|
| `01-recommend.png` | `/` | Course hole selected, **Recommendation** card with TOP PICK and ranked bag |
| `02-live-scorecard.png` | `/` | Active live round with **Scorecard** and multiple players |
| `03-league-standings.png` | `/social/leagues` | League **Standings** tab with podium hero |
| `04-pair-shuffle.png` | `/social/leagues` | Doubles league **Pairs** tab after shuffle |
| `05-profile-badges.png` | `/profile` | **Progress** panel with badges |
| `06-community.png` | `/social/community` | Community with events or playing today |

## Bonus

- `/invite` — full page with **See how Caddy thinks** demo (good for web marketing)
- `/updates` — roadmap columns (Shipped / In progress / Coming soon)
- End-of-round **Round recap link** banner on Home

## Tips

- Use light and dark mode if your store listing supports both.
- Hide personal email in profile screenshots.
- Prefer real course names from your catalog over empty states.
